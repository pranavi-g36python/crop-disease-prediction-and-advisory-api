Link: https://link.springer.com/article/10.1023/A:1010933404324

Description:

Link: https://arxiv.org/abs/1603.02754

Description:
This paper presents the XGBoost algorithm, a high-performance gradient boosting model widely used in industry. It provides the mathematical and algorithmic basis for using XGBoost for complex prediction tasks like crop yield estimation.
